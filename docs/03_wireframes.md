```

- H3: 24px / 32px line-height

本文:
- Font: "Geist Sans"

```

## 共通コンポーネント

### ヘッダー（デスクトップ）
- 左側：ロゴ
- 右側：ナビゲーションメニュー（横並び）
- 背景：半透明のクリーム色
- スクロール時：スティッキー

### ヘッダー（モバイル）
- 左側：ハンバーガーメニュー
- 中央：ロゴ
- ドロワーメニュー：全画面表示

### フッター
- 3カラム構成
  - サイトマップ
  - SNSリンク
  - コピーライト
- 背景：ナチュラルブラウン

### カード型コンポーネント
- 角丸（8px）
- 薄い影
- ホバー時：影を濃く、少し浮き上がる

### ボタン
- Primary: 暖色系オレンジ、角丸
- Secondary: 枠線のみ、透明背景
- ホバー時：色の反転
