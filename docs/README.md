## プロジェクト概要

**サイト名**: Pocket Daké  
**目的**: Vulfpeckの情報を日本のファンに向けて発信・集約  
**技術スタック**: Next.js 15 + TypeScript + Tailwind CSS + Cloudflare Workers  
**リリース予定**: Phase 1 - 即日
